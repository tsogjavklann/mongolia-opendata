# 🇲🇳 Монголын Нээлттэй Өгөгдлийн SQL Хөдөлгүүр

> Монголын нээлттэй өгөгдлийн Power BI — гэхдээ маш хялбар.

[1212.mn нээлттэй өгөгдөл](https://opendata.1212.mn)-ийг SQL-тэй төстэй синтакс ашиглан асуулга хий.
Шуурхай график, хүснэгт — файл татахгүй, гадаад сервергүй.

---

## Боломжууд

- **SQL горим** — мэргэжлийн түвшинд асуулга бичих
- **Хайгч горим** — техник мэдлэггүй хэрэглэгчдэд зориулсан форм
- **Шугаман график** — Хугацаагаар автоматаар үүсгэгддэг, олон цуврал дэмжинэ
- **Өгөгдлийн хүснэгт** — гүйлгэх боломжтой
- **CSV экспорт** — бүх мөрийг татах
- **Холбоос хуваалцах** — асуулга URL-д кодлогдсон
- **Debug самбар** — API payload болон хариуг шалгах
- **Дахин оролдох логик** — 3 удаа экспоненциал хоцрогдолтойгоор дахилт
- **Санах ойн кэш** — 1 цагийн TTL
- **Алдааг зохистой зохицуулах** — 1212.mn унасан ч програм зогсохгүй

## SQL Синтакс

```sql
SELECT *
FROM <хүснэгтийн_id>
WHERE Period BETWEEN 'ЖЖЖЖСС' AND 'ЖЖЖЖСС'
  AND CODE IN ('01', '02')
  AND CODE1 IN ('A')
  AND CODE2 IN ('B')
LIMIT 1000;
```

### Дүрмүүд

| Заалт | Утга |
|-------|------|
| `FROM` | 1212.mn дахь хүснэгтийн ID (жш: `DT_NSO_2400_024V1`) |
| `Period BETWEEN 'ЖЖЖЖСС' AND 'ЖЖЖЖСС'` | Сарын массив үүсгэнэ |
| `Period IN ('202301', '202302')` | Тодорхой хугацааны жагсаалт |
| `CODE IN (...)` | CODE хэмжээсээр шүүх |
| `CODE1 IN (...)` | CODE1 хэмжээсээр шүүх |
| `CODE2..CODE6 IN (...)` | Нэмэлт хэмжээсийн шүүлт
